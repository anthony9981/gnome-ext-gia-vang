## Installation

### Install gnome tweak
```bash
# On Ubuntu 10.04
sudo apt update
sudo apt install gnome-tweak-tool -y
```
### Clone Extension 

```bash
cd ~/.local/share/gnome-shell
git clone https://github.com/anthony9981/gnome-ext-gia-vang.git
```
